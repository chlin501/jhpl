Package de.linearbits.jhpl

  • Class Summary 
    Class Description
    JHPLBuilder<T>
    This class implements a builder for nodes in the source space.
    JHPLNodes<T>
    This class implements methods for working with nodes.
    JHPLSpace<T>
    This class provides methods for mapping between three different spaces for nodes: The source space: this space is meant to provide natural representations of elements, e.g., (A, B, C) The index space: this space represents elements by indices that correspond to the position of each element in the source space.
    JHPLUnsafe
    This class provides methods that may not be safe to use on very large lattices because their complexity grows linearly with the size of the lattice.
    Lattice<T,U>
    This class implements a storage structure for information about elements in very large lattices.
    PredictiveProperty
    This class implements a property that is predictive within the lattice.
